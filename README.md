# STQA-5th-Semester
STQA semester project covering manual testing, automation testing, API testing, defect reporting, and documentation for the BookCart web application.

# Project Description
This repository contains the complete Software Testing & Quality Assurance (STQA) semester project for the STQA course. The project includes manual testing, automation testing, API testing, defect reporting, and final documentation for the BookCart web application.

# Website Under Test
https://bookcart.azurewebsites.net/

# Deliverables Included
- Automation Scripts (Python) with execution video and output report
- API Testing Postman Collection (JSON)
- Defect Reporting Summary
- Complete Project Report
- Test Plan and Test Cases
- Presentation Slides
